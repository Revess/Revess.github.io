SynthSong

The second project I did for University in my second year. The assignment was to create a synthesizer that can play a song from an algorithm. The synthesizer is generated using C++ together with the algorithm for deciding what to play. The algorithm for the notes itself is a simple Markov Chain, in the included example you will be able to edit multiple parameters which are explained on the Github page or in the included video.

Irregular Beat Generator

In this project I used Python to generate an irregular beat based on samples. Each of the three elements of the drumkit (kick, hat, snare) have their own weight where kick is most important, the snare second and the hats fill in the gaps on an irregular basis. Each generated file can also be exported as a midi file.
